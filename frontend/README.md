# Frontend - Дневник успеха

Frontend приложение согласно `rules.md`.

## Структура

```
frontend/
├── src/              # Исходный код
│   ├── js/          # JavaScript модули
│   ├── css/         # CSS файлы
│   └── components/  # Компоненты (если будет фреймворк)
├── static/          # Статические файлы
│   ├── css/         # Скомпилированные CSS
│   ├── js/         # Скомпилированные JS
│   └── images/      # Изображения
└── config/          # Конфигурационные файлы
```

## Текущая структура

Сейчас используется Vanilla JS. В будущем можно перейти на Angular согласно `rules.md`.

## Файлы

- `index.html` - главный HTML файл
- `src/js/` - JavaScript модули (error-handler.js, utils.js, data.js, ui.js, app.js)
- `static/css/styles.css` - стили
- `config/` - конфигурация
